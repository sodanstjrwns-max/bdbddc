해결: src/index.tsx에 www → non-www 301 리다이렉트 코드 추가 완료. 배포 필요.

🔧

수정 완료 #2: 404 페이지가 500 에러를 반환하던 문제

문제: 존재하지 않는 URL 접근 시 500 Internal Server Error 반환 (catch-all 라우트 주석 처리됨)
해결: app.notFound() + app.onError() 핸들러 추가하여 커스텀 404 페이지 + 정상 404 상태코드 반환. 배포 필요.
